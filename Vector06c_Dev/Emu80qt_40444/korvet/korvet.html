Корвет. Краткая справка

Общая информация

После сброса и прохождения тестов без подключенных образов дисков  запускается встроенный Бейсик. Для запуска программ в большинстве случаев следует подключить образ диска в формате .kdi.

Работа с образами дисков

Файлы образов дискет (.kdi) для дисководов A-D подключаются с помощью кнопок на панели инструментов, горячих клавиш Alt-A (Meta-A) для диска «A» и Alt-B (Meta-B) — для диска «B» либо соответстующих пунктов меню. Загрузка с диска A происходит автоматически после сброса. После загрузки операционной системы CP/M можно набрать DIR для вывода списка файлов на диске либо набрать имя программы для ее запуска.

Особенности сброса

По умолчанию в эмуляторе активирован быстрый сброс с помощью клавиш Alt-F11, для быстрого сброса можно также воспользоваться штатным способом Корвета — F1+F11 (просто F11 — всегда полный сброс).

Загрузка с использованием перехватов магнитофонных операций

В эмуляторе реализован перехват магнитофонных операций и работа с файлами формата .cas. Чаще всего данный формат применяется в Бейсике. Для загрузки файлов служит команда Бейсика «LOAD"CAS:"».

Соответствие клавиш и раскладки клавиатуры

Для ПК «Корвет» рекомендуется использовать «Умную» раскладку клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка и регистр переключается средствами ОС, в которой запущен эмулятор. Раскладки клавиатуры приведены ниже:
