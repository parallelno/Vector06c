Орион-128. Краткая справка

Общая информация

После запуска данной конфигурации с ROM-диска запускается оболочка ORDOS. Запуск файлов возможен с одного из 4 дисков - A, B, C или D. Диск A представляет собой ROM-диск, диски B, C и D — RAM-диски.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы Ориона .rko, .bru, .ord можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

Файлы образов лент .rko можно загрузить с помощью загрузчика CH4¤, который нужно запустить с ROM-диска A. Переключение между дисками в нем производится стрелками влево и вправо, для чтения файла нужно нажать Enter на пункте «ВВОД:». Выход в ORDOS - F4. Для загрузки файлов образов лент в формате RK (.rk?) служит программа CHR¤.

Работа с образами дисков

Файлы образов дисков (.ord) подключаются с помощью горячих клавиш Alt-A (Meta-A) для диска «A» и Alt-B (Meta-B) для диска «B». Можно также использовать соответствующие пункты меню или кнопки на панели инструментов. Для загрузки файлов с дисков ORDOS в RAM-диск можно использовать программы ALT33¤ либо ATLAS¤. Выход из этих программ – «Q». Для загрузки OS-DOS с диска служит программа DOS¤. Для загрузки с дисков с ОС CP/M нужно использовать программу BOOT¤.

Соответствие клавиш и раскладки клавиатуры
