Общая информация

Образы дисков с программами для этого компьютера можно скачать на сайте http://bashkiria-2m.narod.ru/ (раздел «Файлы»)

Работа с образами дисков

Файлы образов дискет (.cpm) для дисководов подключаются с помощью кнопок на панели инструментов, горячих клавиш Alt-A (Meta-A) для дисков «A/B» и Alt-B (Meta-B) — для дисков «C/D» либо соответстующих пунктов меню. После сброса и появления мигающего курсора необходимо нажать на любую клавишу для загрузки с обраа диска. Каждый образ диска представлен двумя буквами дисков, соответствующим двум сторонам дискеты. Например, при подключении первого образа диска две его стороны он будет доступны как диски A и B.

Соответствие клавиш и раскладки клавиатуры
