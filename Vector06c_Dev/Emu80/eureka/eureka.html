Eureka. Краткая справка

Общая информация

ПК «Eureka» является клоном «Специалиста», обладающим расширенными возможностями. Однако встроенное ПО несовместимо со Специалистом, поэтому приемы работы с этим ПК отличаются.

После выбора данной конфигурации начальный загрузчик выдает приглашение «ВВОД?». Далее возможны следующие действия:

Загрузка Монитора ПК «Eureka»

Для загрузки Монитора ПК «Eureka» из дополнительного ROM-диска необходимо нажать F2. Данный Монитор работает с файлами ленты в формате РК-86.

Запуск Монитора ПК «Специалист» из дополнительного ПЗУ

Для запуска Монитора Специалиста нужно нажать клавишу F3, после чего отменить запроси имени файла, нажать одну из клавиш Fx и затем Enter. В этом режиме можно загружать и запускать файлы, предназначенные для ПК «Специалист».

Загрузка rke-файла по сбросу

Для загрузки rke-файла нужно нажать одну из клавиш F6-F11, после чего выбрать rke-файл.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rke» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

Перехваты операций с магнитофоном реализованы для Монитора Специалиста, Монитора Eureka и начального загрузчика

Для загрузки rke-файла после сброса в ответ на приглашение "ВВОД?" нужно нажать одну из клавиш Esc, Tab, F6-F9, F12 после чего выбрать rke-файл.

В Мониторе Eureka в ответ на приглашение нужно набрать «I<Enter>» и выбрать файл .rke. После загрузки запуск обычно осуществляется с помощью директивы G Монитора («G<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «G100<Enter>». Этим способом можно также загружать .wav и .csw-файлы.

Загрузка в Мониторе Специалиста производится аналогично, но вместо директивы «I» для загрузки .rks-файлов используется директива «R».

Соответствие клавиш и раскладки клавиатуры

По умолчанию для ПК Eureka установлена Qwerty-раскладка клавиатуры, с однозначным соответствием клавиш ПК клавишам PC. «Умная» smart-раскладка работает в Мониторе Eureka, но не работает в Мониторе «Специалист».
