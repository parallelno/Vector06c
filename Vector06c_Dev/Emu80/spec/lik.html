Лик. Краткая справка

Общая информация

ПК «Лик» является клоном «Специалиста», обладающим, однако, другим встроенным ПО. По этой причине приемы работы с этим ПК отличаются.

Загрузка файлов с помощью встроенного загрузчика

Двоичные файлы образов ленты «.rks» можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также просто перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

В Мониторе ПК Лик в ответ на приглашение нужно набрать «I=<Enter>» и выбрать файл .rks. После загрузки запуск обычно осуществляется с помощью директивы J Монитора («J=<Enter>»). В случае запуска не с нулевого адреса в директиве G можно указать адрес запуска, например: «J=100<Enter>». Возможно также загружать и запускать файлы одной директивой G («G=<Enter>») Этим способом можно также загружать .wav и .csw-файлы.

Запуска втроенного бейсика

Для запуска встроенного Бейсика служит директива «B» Монитора. Для загрузки программ на бейсике служит команда MLOAD.

Соответствие клавиш и раскладки клавиатуры

По умолчанию для ПК Лик установлена Qwerty-раскладка клавиатуры, с однозначным соответствием клавиш ПК клавишам PC. «Умная» smart-раскладка на данный момент реализована не полностью (не работают буквы кириллицы).
