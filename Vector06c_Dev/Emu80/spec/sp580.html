Специалист SP-580. Краткая справка

Общие сведения

«Специалист SP-580» по архитектуре в основном совпадает с обычным «Специалистом». Среди отличий следует отметить наличие таймера ВИ53 для воспроизвелдения звука (схема подключения аналогична схеме подключения таймера в Радио-86РК), а также отсутствие цветовых расширений. Эмулятор по умолчанию использует черно-белый режим для SP-580, переключение в цветной режим, аналогичный обычному «Специалисту» возможно, однако программы для SP-580 цвет не используют. Также SP-580 имеет довольно много различий во встроенном ПО, поэтому работа программ для обычного «Специалиста» на SP-580 не гарантируется.

Загрузка файлов

SP-580 использует собственный формат обмена с магнитофоном. На данный момент нет форматов файлов образов лент для этого компьюиера, возможна загрузка с использованием звуковых wav-файлов (или csw). После запуска платформы в ответ на сообщение «DDT.SP580» следует нажать на любую клавишу для выхода в Монитор. Для загрузки файлов служит директива Монитора «I», после ее ввода нужно открыть wav-файл с загружаемой программой (комбинация клавиш Alt-W, либо соответсвтуующий пункт меню, либо ннопка на панели инструментов). Для ускорения загрузки рекомендуется в нстройках включить опцию ускорения работы эмулятора при воспроизведении wav-файлов.

Загрузка файлов RKS

SP-580 поддерживает также загрузку файлов в формате RKS обычного «Специалиста» без имени. Для этого перед директивой «I» следует предварительно выполнить директиву «@» (в QWERTY-раскладке эмулятора это кнопка «`~» слева от «1»). При этом, если включен режим перехвата магнитофонных операций, будет выведен диалог выбора файла RKS. Запуск загруженного файла осуществляется далее с помощью директивы «G». Также RKS-файлы можно загружать с помощью встроенного в эмулятор загрузчика (меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов). Можно также просто перетащить нужный файл в окно эмулятора.

Соответствие клавиш и раскладки клавиатуры

Для Специалиста SP-580 рекомендуется использовать «Умную» раскладку клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор.
