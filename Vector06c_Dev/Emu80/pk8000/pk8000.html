ПК-8000. Краткая справка

Общая информация

После выбора данной конфигурации из ПЗУ запускается встроенный Бейсик. Возможные дальнейшие действия описаны ниже.

Загрузка файлов с помощью встроенного загрузчика

Большинство двоичных файлов ПК8000, а также токенизированные файлы Бейсика (.cas) можно загружать с помощью пункта меню «Файл -> Загрузить и выполнить», комбинации клавиш Alt-F3 (Meta-F3), либо соответствующей кнопки на панели инструментов. Можно также перетащить нужный файл в окно эмулятора.

Загрузка с использованием перехватов магнитофонных операций

Для загрузки двоичных файлов служит команда Бейсика «BLOAD"имя",R», токенизированных файлов Бейсика — «CLOAD"имя"», текстовых файлов Бейсика — «LOAD"имя"».

Работа с образами дисков

Для работы с дисководом следует выбрать соответствующую конфигурацию.

Файлы образов дискет (.fdd) подключаются с помощью горячих клавиш Alt-A (Meta-A) для диска «A» и Alt-B (Meta-B) — для диска «B. Можно также использовать соответствующие пункты меню или кнопки на панели инструментов. Загрузка с диска происходит автоматически после сброса.

Работа с жестким диском (CF-картой)

Для работы с жестким диском (CF-картой) следует выбрать соответствующую конфигурацию. При этом автоматически подключается поставляемый с эмулятором образ CF-карты объемом 2 Мб, содержащий большинство разработанных для ПК8000 программ (кроме программ на Бейсике). Загрузка с диска происходит автоматически после сброса. При необходимости путь к файлу образа диска может быть изменен в конфигурационном файле.

Соответствие клавиш и раскладки клавиатуры

Для ПК8000 рекомендуется использовать «Умную» раскладку клавиатуры, ввод буквенно-цифровых символов при этом производится как обычно, раскладка переключается средствами ОС, в которой запущен эмулятор. Раскладки клавиатуры приведены ниже:
